# XFT Documentation

This site provides an overview of the XFT protocol and its tokenized asset platform.  
XFT enables onchain trading, issuance, and settlement of stocks, bonds, and stablecoins.

## Getting Started

- Technical specs, contract roles, and developer guides are included in these docs.
- Use these resources to build, integrate, or audit XFT smart contracts and APIs.
